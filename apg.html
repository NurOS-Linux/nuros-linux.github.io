AnmiTali Package (apg)

Пакет для проекта NurOS, сделанный для tulpar пакетника и основанный на .tar.xz

Описание apg

apg (AnmiTali Package) - это пакет для проекта NurOS, специально разработанный для использования с tulpar пакетником. Он основан на формате .tar.xz и имеет следующую структуру:

├── metadata.json
├── scripts/
│   ├── preinstall
│   ├── postinstall
│   ├── preremove
│   └── postremove
├── data/
│   └── ...
└── md5sums (пожеланию разработчика)

Содержимое apg пакета

  • metadata.json: Файл с метаданными пакета.
  • scripts/: Директория со скриптами для различных этапов установки и удаления пакета.
    • preinstall: Скрипт, выполняющийся перед установкой пакета.
    • postinstall: Скрипт, выполняющийся после установки пакета.
    • preremove: Скрипт, выполняющийся перед удалением пакета.
    • postremove: Скрипт, выполняющийся после удаления пакета.
  • data/: Директория с данными пакета.
  • md5sums: (по желанию разработчика) Файл с контрольными суммами MD5 для проверки целостности файлов.
