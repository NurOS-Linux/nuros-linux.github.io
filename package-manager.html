Пакетный менеджер tulpar

Мощный и современный пакетный менеджер для NurOS

О tulpar

tulpar - это универсальный и мощный пакетный менеджер, разработанный специально для NurOS. Он позволяет легко управлять программным обеспечением, устанавливать, обновлять и удалять пакеты.

Как работает Tulpar

Tulpar работает с пакетными репозиториями, которые хранят бинарные и исходные пакеты. Когда пользователь выполняет команду установки или удаления, Tulpar автоматически:

  1. Проверяет доступные репозитории
  2. Скачивает необходимые файлы и зависимости
  3. Устанавливает их в систему

Во время установки или удаления пакетов все зависимые компоненты обрабатываются автоматически, чтобы обеспечить функциональность программы.

Команды для управления пакетами

# Установка пакета
tulpar install <имя_пакета>

# Удаление пакета
tulpar remove <имя_пакета>

# Обновление конкретного пакета
tulpar update <имя_пакета>

# Установка из локального файла .apg
tulpar -i <package.apg>

# Установка из удаленного репозитория/git
tulpar -i <repo-url/git-url>

# Обновление системы
tulpar upgrade

# Информация о пакете
tulpar info                    # Краткий список всех пакетов
tulpar info <имя_пакета>     # Подробная информация о пакете

# Поиск пакетов
tulpar search <запрос>

Команды для управления репозиториями

# Добавление репозитория
tulpar-repo -a <repository-url>

# Удаление репозитория
tulpar-repo -r <repository-url>

Установка Tulpar

# Сборка из исходников
git clone https://github.com/nuros-linux/tulpar.git
cd tulpar
make
sudo make install

Конфигурация

Файл конфигурации по умолчанию находится по адресу /etc/tulpar/tulpar.conf

Файл списка репозиториев находится по адресу /etc/tulpar/repolist.json

Распакованные пакеты находятся по адресу /var/lib/tulpar/packages/
