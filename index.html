О системе

NurOS — это инновационный независимый GNU/Linux дистрибутив, созданный с нуля для обеспечения максимальной производительности и безопасности.

Основные задачи:

  • Информирование пользователей о дистрибутиве и его компонентах
  • Предоставление доступа к документам, пакетам и поддержке
  • Связь с разработчиками через Telegram-каналы
  • Безопасное скачивание дистрибутива через Telegram

Tulpar - Менеджер пакетов

Основные возможности:

  • Быстрая установка и обновление пакетов
  • Интеграция с APG форматом
  • Автоматическое разрешение зависимостей
  • Система контроля версий пакетов

Maintainer: GoldenVadim

APG - Формат пакетов

Характеристики:

  • Современный формат упаковки
  • Оптимизированная структура метаданных
  • Встроенная система верификации
  • Поддержка скриптов пре/пост установки

Основные компоненты

UPKGT

Инструмент для улучшения пользовательского опыта и управления пакетами

Astrum

Компонент системного управления и администрирования

Tex

Набор инструментов для системного администрирования

DDE

Современное окружение рабочего стола

Ata

Архиватор на основе Zstd (AnmiTali Archivator)
